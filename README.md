<br>
An Android application that detects and identifies 8 different types of Bangladeshi vehicles in real-time using machine learning. 
The app is powered by a TensorFlow Lite model trained using Teachable Machine. It supports both real-time camera capture and 
gallery image selection, making vehicle detection fast, accurate, and user-friendly.
<br>
<br>

## Features:
* Real-time Detection: Identify 8 vehicle types: Rickshaw, CNG, Van, Car, Bicycle, Bike, Leguna, and Bus
* Dual Input Methods:
  Camera Capture: Take photos in real-time
  Gallery Selection: Choose from existing images
* High Accuracy: ML-powered classification with confidence scores
* User-Friendly Interface: Clean and intuitive UI with visual feedback

<br>

## Prerequisites:
* Android Studio (latest version)
* Android SDK 21+
* Java Development Kit (JDK) 11+
* Android device/emulator with camera
* Install the required packages using pip


<br>

## Technical Specifications: 
* Programming Language: Java
* ML Framework: TensorFlow Lite
* Minimum SDK: Android 5.0 (API 21)
* Target SDK: Android 13 (API 33)
* Permissions: Camera and Gallery access
